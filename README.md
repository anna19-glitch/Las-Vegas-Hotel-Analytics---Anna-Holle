# 5-Star or Flop? What Drives Hotel Reviews on the Las Vegas Strip?
In this project, you will find an exploratory analysis of Hotel Reviews on the Las Vegas Strip. 
## Goal
The goal of this project was to determine what drives hotel reviews on the Las Vegas Strip. A number of factors are examined to determine if there is anything that contributes to a good or bad review of a hotel. This analysis may be useful to entrepreneurs looking to start a successful hotel business, or it may be useful to hotel owners looking to improve their services.
## Folder Structure
This project, contains 2 notebooks. Part 1 contains the basic data exploration and Part 2 contains the answers to the business questions. In addition, the plots produced by the code are stored in the plots folder. The 3 page "Hotel Analytics Report" is included in the report folder.
## How to Run the Program
Simply open the two ipynb files, ensure the data is downloaded from the repository, and run the code. 
## Outputs
The code in this project produces mostly print outputs of answers to the questions, and produces 3 plots.
